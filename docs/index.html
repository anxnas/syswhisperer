Обзор

SysWhisperer - это утилита для анализа системной информации, которая предоставляет подробные данные о вашем компьютере, включая процессор, память и видеокарты. Поддерживает работу на Windows и Linux.

Основные возможности:

  • Получение информации о процессоре (количество ядер, частота, загрузка)
  • Мониторинг оперативной памяти
  • Поддержка видеокарт NVIDIA, AMD и Intel
  • Экспорт данных в JSON
  • Кроссплатформенность

Установка

Windows

  1. Скачивание

    Скачайте последнюю версию SysWhisperer для Windows (файл syswhisperer.exe) со страницы релизов.

  2. Запуск

    Запустите программу из командной строки:

    syswhisperer.exe --version

  3. Опционально: Добавление в PATH

    Для удобства использования можно добавить путь к программе в системную переменную PATH:

    1. Создайте папку для программы, например: C:\Program Files\SysWhisperer
    2. Переместите exe файл в эту папку
    3. Откройте "Система" → "Дополнительные параметры системы" → "Переменные среды"
    4. В разделе "Системные переменные" найдите PATH
    5. Нажмите "Изменить" и добавьте путь к папке
    6. После этого программу можно запускать просто как syswhisperer.exe из любой папки

Linux

  1. Скачивание

    Скачайте последнюю версию SysWhisperer для Linux (файл syswhisperer) со страницы релизов.

  2. Установка

    chmod +x syswhisperer
sudo mv syswhisperer /usr/local/bin/syswhisperer

  3. Проверка установки

    syswhisperer --version

Установка из исходного кода

  1. Клонирование репозитория

    git clone https://github.com/anxnas/syswhisperer.git
cd syswhisperer

  2. Установка зависимостей

    pip install -r requirements.txt

  3. Запуск

    python main.py --version

Использование

Командная строка

# Базовое использование
syswhisperer

# Только информация о GPU
syswhisperer --no-cpu --no-memory

# Экспорт в JSON
syswhisperer --export result.json

Параметры командной строки

Параметр Описание
--version Показать версию программы
--no-cpu Не показывать информацию о процессоре
--no-memory Не показывать информацию о памяти
--no-gpu Не показывать информацию о видеокарте
--export FILE Экспортировать данные в JSON файл
